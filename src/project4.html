CHIP-8 Interpreter

C++, SDL2, CMake

Overview

CHIP-8 was first introduced in the 1970s. Many famous classic games like Tetris, Pac-Man, Pong, etc. were ported to it.

This project is a CHIP-8 interpreter, and not an emulator. This is because the CHIP-8 isn't a physical machine being emulated, but rather a virutal machine running CHIP-8 programs.
