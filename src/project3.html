Country Web API

Javascript, HTML/CSS

(rough)



Goal

The goal of this project was to create a Web API through which users could quickly access basic information about any country.

Target userbase

The target user for this site is someone who may be interested in a particular country, whether it be for travelling purposes or general curiousity.
The site displays info about a country's population, weather, timezone, and national languages, which are particularly useful for anyone who may be interested in travelling.

How it works

The site uses HTML/CSS for its simple frontend. Javascript is used to access the APIs which display the info.
Postman was used to test the various API calls that were made.

The APIs used are accessed via a fetch(url) call, which receives the URL of the API containing its respective API key as a parameter.
  • REST Countries is used to retrieve basic information about whichever country the user searches for.
  • MapBoxGL is used to pinpoint the location of the country in a Google Earth-style map.
  • Unsplash is used to get images representing the country that is searched for as well as its national flag.
  • TimeZoneDB is used to get the local time at the capital city of the country that is searched for.
  • OpenWeatherMap is used to get the weather of the country that is searched for.
