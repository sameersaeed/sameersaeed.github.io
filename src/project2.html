Stock / Cryptocurrency Web API

Javascript, HTML/CSS

(rough)



Goal

The goal of this project was to create a Web API through which users could quickly access data about any cryptocurrency or stock listed on Alpha Vantage's stock API.

Target userbase

The target user for this site is anyone who is interested in a stocks or cryptocurrencies, whether they are looking them up for potential investments or other reasons such as out of interest of price history.

How it works

The site uses HTML/CSS for its simple frontend. Javascript is used to access the APIs which display the info.
Postman was used to test the various API calls that were made.
The APIs used are accessed via a fetch(url) call, which receives the URL of the API containing its respective API key as a parameter.

The main API that is used is Alpha Vantage's stock API, which retrieves price data about the stock/cryptocurrency that the user searches for.
A graphing API, ApexCharts, is also used in order to display this data in a graphical format.

Future plans

  • Implement new features such as an account system so that users can track various stocks/cryptocurrencies of their choosing
  • Display more info, such as latest news about the stock/crypto, projections, expert opinions, etc.
  • Improve site design: create a name, logo, theme, etc.
